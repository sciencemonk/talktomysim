# SIM - Agentic Storefront Platform

Welcome to **SIM**, the world's first agentic storefront platform that revolutionizes e-commerce by combining AI-powered sales agents with seamless x402 cryptocurrency payments.

## What is SIM?

SIM transforms traditional online stores into intelligent, conversational commerce experiences. Instead of static product catalogs, your customers interact with AI agents that understand their needs, recommend products, and guide them through the purchasing journey‚Äîall while accepting payments via the x402 protocol on Solana.

## How It Works

1. **Create Your Store**: Sign up with your Coinbase account to get an embedded wallet
2. **Add Products**: Upload your products with images, descriptions, and prices
3. **Configure Your AI Agent**: Customize how your agent interacts with customers
4. **Go Live**: Share your store link and start making sales

## Technology Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase Edge Functions
- **AI**: Lovable AI Gateway (Google Gemini & OpenAI)
- **Payments**: x402 Protocol on Solana
- **Wallets**: Coinbase CDP (Embedded Wallets)

## Getting Started

### Prerequisites
- Node.js 18+ installed
- A Coinbase account (for embedded wallet)
- Supabase project (for backend)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/sim.git

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start the development server
npm run dev
```

### Environment Variables

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_key
VITE_CDP_PROJECT_ID=your_coinbase_project_id
```
